## 5. 自動化回歸（工程，發 PR 前必跑）

| 項目 | 指令 | 通過 |
|------|------|------|
| Lint + 單元/API | `npm run test:regression` | □ |
| Vitest 全量 | `npm test` | □ |
| E2E 煙霧（需可 build；會自動起 server 或使用已跑著的 dev） | `npm run test:e2e` | □ |
| Build | `npm run build` | □ |

**說明：**

- `tests/unit/`：權限、`getApiError`、錯誤回應、附件驗證等。  
- `tests/api/`：例如 `GET /api/health` 路由。  
- `tests/e2e/`：Playwright — health API、`/login`、`/unauthorized` 煙霧。  
- 本機 E2E：預設會 `npm run build && npx next start`（首次較久）；若已在 `npm run dev`，可另開終端機只跑測試：  
  `set PLAYWRIGHT_SKIP_WEBSERVER=1`（Windows）或 `PLAYWRIGHT_SKIP_WEBSERVER=1`（Unix）後  
  `set PLAYWRIGHT_BASE_URL=http://127.0.0.1:3000` 再 `npm run test:e2e`。
